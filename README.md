# Mon Portfolio

Portfolio personnel inspiré du design de [Brittany Chiang](https://brittanychiang.com), construit avec React + Vite.

## Stack

- **React 18** + **Vite**
- **CSS Modules** pour les styles
- **IntersectionObserver** pour les animations au scroll

## Setup

```bash
# Installer les dépendances
npm install

# Lancer le serveur de dev
npm run dev

# Build pour la production
npm run build

# Prévisualiser le build
npm run preview
```
